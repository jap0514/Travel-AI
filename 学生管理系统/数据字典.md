

#### 1、**sys_user（用户表）**

| phone           | VARCHAR(20)  | Y    | -      | -                 | 手机号              |

| status          | TINYINT      | Y    | -      | 1                 | 状态：0-禁用 1-正常 |
| last_login_time | datetime     | Y    | -      | -                 | 最后登录时间        |

| created_time    | datetime     | Y    | -      | CURRENT_TIMESTAMP | 创建时间            |
| updated_time    | datetime     | Y    | -      | CURRENT_TIMESTAMP | 更新时间            |
